# Acknowledgments

## Third-Party Software

Pandowdy incorporates the following third-party software components:

### CiderPress II

**Author:** Andy McFadden (faddenSoft)  
**Website:** https://ciderpress2.com/  
**Repository:** https://github.com/fadden/CiderPress2  
**License:** Apache License 2.0  

Pandowdy uses the following libraries from CiderPress II:
- **DiskArc** - Disk image and file archive management
- **FileConv** - File format conversion utilities
- **CommonUtil** - Common utility functions

CiderPress II is an excellent tool for working with disk images and file archives 
associated with vintage Apple computers. We are grateful to Andy McFadden for 
making this high-quality software available under the Apache 2.0 license.
